# 基于 30–300MHz 频谱响应的液体识别系统

# 需求分析文档

---

# 1. 项目背景

目前已经能够通过射频/电磁响应测量系统，对不同液体在：

```text
30 MHz ~ 300 MHz
```

频率范围内进行扫描，并获得对应的频谱响应数据。

实验结果表明：

不同液体在部分频段会表现出明显不同的响应特征，包括：

* 峰值位置差异
* 峰值幅度差异
* 谷值深度差异
* 高频段增长趋势差异
* 曲线斜率变化
* 特征频段形态差异

虽然整体频谱曲线存在一定重叠，但在某些关键频段内，不同液体已经表现出较好的可分性。

因此，希望构建一个：

# “基于频谱响应的液体识别系统”

实现：

* 液体频谱分析
* 关键频段提取
* 特征建模
* 液体自动分类
* 未知液体识别

---

# 2. 项目目标

本项目目标是：

# 通过 30–300MHz 范围内的频谱响应，实现未知液体自动识别。

---

关键是模型的构建，构建一个模型，数据输入，得到对应的液体名称

## 输入

系统输入：

```text
30–300 MHz 频谱响应数据
```

数据形式：


| Frequency (MHz) | Amplitude (dB) |
| --------------- | -------------- |
| 30              | 0.52           |
| 31              | 0.61           |
| ...             | ...            |
| 300             | 20.14          |

---

## 输出

系统输出：

```text
液体类别 + 预测置信度
```

例如：

```text
Prediction:
NaCl

Confidence:
91.2%
```

# 4. 数据需求分析

# 4.1 数据来源

所有实验数据统一保存在：

```text
data/
```

目录中。

数据包括：

* 不同液体频谱数据

* 不同实验条件结果

系统需要能够直接读取：

```text
data/
```

中的数据进行分析。

---

# 4.2 当前液体类别

当前主要研究液体包括：

* Water
* NaOH
* HCl
* NaCl
* Ethanol
* Glycerol

后续支持扩展更多液体。

---

# 4.3 数据特点

当前数据具有以下特点：

---

## 多峰结构明显

曲线中存在多个：

* 峰值
* 谷值
* 局部突变

说明：

液体在不同频段存在不同介电响应。

---

## 高频趋势差异明显

高频区域：

```text
270–300 MHz
```

不同液体存在：

* 上升速度差异
* 峰值高度差异
* 恢复速度差异

具备较强分类价值。

---

## 数据存在噪声

低频区域存在：

* 高频抖动
* 测量误差
* 小幅随机波动

需要进行：

* 平滑
* 去噪
* Normalize

处理。

# 9. 当前阶段核心重点

当前阶段最重要的任务：

不是复杂深度学习模型。

而是：

# “关键频段分析 + 特征稳定性分析”

原因：

当前实验结果已经证明：

```text
不同液体在特定频段已经具有明显可分性
```

因此：

系统成功的核心在于：

* 找到最稳定特征区
* 提取最有效频谱特征
* 建立稳定分类规则

# “基于 30–300MHz 频谱响应的未知液体自动识别系统”
